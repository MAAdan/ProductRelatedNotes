# Product Related Topics

This repository is a live document for collecting notes about product-related books, articles, and other information. It will keep growing over time as new ideas, references, and learnings are added.

## Current Focus

The notes collected so far focus on product management foundations, especially around roadmaps and prioritisation.

At the moment, the material explores how product roadmaps should support collaboration, how ideas can be evaluated against user and business needs, and how structured thinking can improve prioritisation. It also touches on practical tools such as PRDs and goal-setting frameworks like SMART goals, CLEAR goals, and OKRs.

The detailed notes currently live in [ProductManagementNotes.md](/Users/migada/ProductRelatedTopics/ProductManagementNotes.md) and this repository will continue to expand as more sources and reflections are added.
